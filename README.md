# ArrayUtility
Some tested utility for Array

## Array.prototype.removeByIndex()

```javascript
var arr = ['a', 'b', 'c', 'd', 'e'];

arr.removeByIndex(2);
console.log(arr); // ['a', 'b', 'd', 'e']
```
## Array.prototype.remove()

```javascript
var arr = ['a', 'b', 'c', 'd', 'e'];

arr.remove('c');
console.log(arr); // ['a', 'b', 'd', 'e']
```
